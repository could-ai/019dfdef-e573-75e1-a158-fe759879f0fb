# تقييم المخيم الصيفي 🏕️

تطبيق Flutter مخصص لحساب الجدوى المالية وتتبع أهداف المخيم الصيفي.

## نظرة عامة
تم تصميم هذا التطبيق بناءً على المعطيات المالية الخاصة بالمخيم الصيفي، حيث يقوم بحساب الإيرادات المتوقعة من الحصص المختلفة، يخصم منها المصاريف الثابتة، ويحسب الربح أو الخسارة الصافية، بالإضافة إلى تتبع هدف تسجيل 21 تلميذاً.

## المعطيات المالية 💰
- **الحصص والإيرادات:**
  - الحصة الأولى: 25,000 د.ج (لمدة أسبوعين)
  - الحصة الثانية: 23,000 د.ج (لمدة أسبوعين)
  - الحصة الثالثة: 20,000 د.ج (لمدة أسبوعين)

- **المصاريف الثابتة:**
  - الوجبات الغذائية: 34,650 د.ج
  - مصاريف أخرى: 10,000 د.ج
  - مصاريف الرحلة: 3,000 د.ج

- **الهدف المطلوب:**
  - تسجيل 21 تلميذ على الأقل.

## المميزات ✨
- لوحة تحكم تفاعلية لحساب الإيرادات.
- تعديل عدد التلاميذ في كل حصة بشكل ديناميكي لمعرفة الأرباح.
- عرض مرئي واضح للمصاريف والإيرادات والربح الصافي.
- دعم كامل للغة العربية والاتجاه من اليمين إلى اليسار (RTL).
- تصميم متجاوب يعمل بشكل ممتاز على الهواتف والأجهزة اللوحية وسطح المكتب.

## المتطلبات التقنية 🛠️
- Flutter SDK (الإصدار الأخير الموصى به)
- Dart SDK

## تشغيل التطبيق ▶️
لتشغيل التطبيق على جهازك المحلي:
```bash
flutter pub get
flutter run
```
